# Card Github

# Sobre o projeto

O projeto é uma aplicação web no formato SPA (Single Page Application) que consome a API do GitHub para trazer algumas informações sobre o perfil pesquisado. Ele foi desenvolvido com finalidade didática.

A aplicação foi apresentada em uma aula disponível no canal do YouTube da [DevSuperior](https://www.youtube.com/devsuperior "YouTube da DevSuperior").

Ela consiste em uma página web com mecanismo de pesquisa para perfis do Github. Exibindo para o usuário informações sobre o perfil pesquisado, tais como: Nome, Localização e Site.
